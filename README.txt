ESCOLA INSTRUCTIVA — PREVIA DO SITE NOVO
==========================================

Como abrir:
- Dê duplo clique em "index.html" para abrir no navegador.
- Navegue normalmente pelos links (Cursos, cada curso individual etc). Tudo funciona offline.

Estrutura de pastas:
- index.html            -> página inicial
- cursos.html            -> catálogo com os 35 cursos, organizados por categoria
- cursos/*.html           -> uma página para cada um dos 35 cursos
- css/style.css           -> estilo visual compartilhado por todas as páginas
- assets/fonts/           -> fontes usadas no site (funcionam offline)
- assets/img/             -> logo da Instructiva

O que ainda é preciso plugar antes de ir pro ar:
1. Trocar os links "Comprar agora" (hoje apontam para "#") pelos links reais de
   checkout de cada curso na Hotmart.
2. Trocar a foto do Prof. Celso (hoje só um círculo com as iniciais "CM").
3. Trocar os depoimentos de exemplo por depoimentos reais de alunos.
4. Revisar os preços — foram herdados do catálogo que já tínhamos montado.
5. Publicar em algum lugar (ex: Railway) e apontar o domínio escolainstructiva.com.br pra lá.
